## CRITICAL RULES (NON-NEGOTIABLE)

### RULE 1: ZERO EM DASHES (MANDATORY)

⛔ ABSOLUTE BAN ON EM DASHES ⛔

==============================

The em dash character (---) is FORBIDDEN in all quiz copy.

WHY: Em dashes create a specific "AI writing" fingerprint that

reduces trust and credibility. Human copywriters rarely use them.

VERIFICATION REQUIRED:

Before delivering ANY quiz funnel, run:

grep -c "---" filename.html

If the count is ANYTHING other than 0, you have FAILED.

Rewrite ALL instances immediately.

APPROVED ALTERNATIVES:

✅ Use commas for clauses

✅ Use periods for separate sentences

✅ Use parentheses for asides

✅ Use colons for introductions

✅ Use semicolons for related thoughts

WRONG: "Our quiz helps you discover your perfect solution---and it only
takes 2 minutes."

RIGHT: "Our quiz helps you discover your perfect solution. It only
takes 2 minutes."

RIGHT: "Our quiz helps you discover your perfect solution (it only
takes 2 minutes)."



**MANDATORY FINAL CHECK:** Before presenting ANY quiz to the user, grep
the HTML file for em dashes. Zero tolerance.

## PSYCHOLOGY FOUNDATION

### The Barnum Effect

**Concept:** Vague statements feel personally accurate\
**Application:** Craft quiz results that feel customized even with
general insights\
**Example:** "You're someone who values efficiency but sometimes
struggles with follow-through" (applies to 80%+ of people but feels
personal)

### The Zeigarnik Effect

**Concept:** Uncompleted tasks create tension that demands resolution\
**Application:** Open loops throughout the quiz drive completion\
**Example:** "Based on your first answer, you might be a [Type], but
let's confirm with question 2..."

### The Self-Serving Bias

**Concept:** People prefer information that makes them look good\
**Application:** Frame all results positively, even "negative" types\
**Example:** Don't say "You're disorganized." Say "You're
creatively spontaneous and thrive in flexible environments."

### The IKEA Effect

**Concept:** Effort creates value and ownership\
**Application:** Make the quiz engaging enough to feel worth completing\
**Example:** Use 7-10 questions (sweet spot for effort-value balance)

### Commitment & Consistency

**Concept:** First action creates momentum for subsequent actions\
**Application:** First question answered = 80%+ completion rate\
**Example:** Make question 1 easy, interesting, and non-threatening

## QUIZ ARCHETYPES (Choose One)

### 1. PERSONALITY QUIZ

**Format:** "What Type of [X] Are You?"\
**Best For:** Awareness stage, email list building, segmentation\
**Result Types:** 4-6 distinct personality types\
**Example:** "What's Your Marketing Personality?"

**Psychology:** Taps into identity and self-discovery\
**Completion Rate:** 75-85% (highest of all types)\
**Conversion Path:** Personality → Tailored solution recommendation

### 2. ASSESSMENT QUIZ

**Format:** "What's Your [X] Score?"\
**Best For:** Problem-aware stage, demonstrating need\
**Result Types:** Scoring scale with ranges\
**Example:** "What's Your Ad Account Health Score?"

**Psychology:** Creates urgency through gap awareness\
**Completion Rate:** 65-75%\
**Conversion Path:** Low score → Solution to improve

### 3. RECOMMENDATION QUIZ

**Format:** "Which [X] is Right for You?"\
**Best For:** Solution-aware stage, product comparison\
**Result Types:** Specific product/service recommendations\
**Example:** "Which Unicorn Marketer is Perfect for Your Business?"

**Psychology:** Reduces decision paralysis\
**Completion Rate:** 70-80%\
**Conversion Path:** Recommendation → Direct booking/purchase

### 4. DIAGNOSTIC QUIZ

**Format:** "What's Blocking Your [Desired Outcome]?"\
**Best For:** Problem identification, consultative selling\
**Result Types:** Primary obstacle with solution path\
**Example:** "What's Killing Your ROAS?"

**Psychology:** Problem agitation + solution relief\
**Completion Rate:** 60-70%\
**Conversion Path:** Diagnosis → Targeted solution

### 5. KNOWLEDGE QUIZ

**Format:** "How Much Do You Really Know About [X]?"\
**Best For:** Educational content, authority building\
**Result Types:** Score-based with educational results\
**Example:** "Meta Ads Mastery Quiz: Beginner to Expert"

**Psychology:** Curiosity + learning validation\
**Completion Rate:** 55-65% (lowest but highest engagement quality)\
**Conversion Path:** Knowledge gap → Training/expertise

### PHASE 4: QUESTION DESIGN (30-40 min)

**4.1 Question Quantity**

Optimal: 7-10 questions

Minimum: 5 questions

Maximum: 12 questions

Completion by question count:

5 questions = 85% completion

7-10 questions = 75-80% completion

12+ questions = drops to 60-65%



**4.2 Question Sequencing**

Question 1: Easy, non-threatening, intriguing

Question 2-3: Persona-defining (biggest impact on result)

Question 4-6: Depth and specificity

Question 7-9: Preference and style

Question 10 (optional): Confirmation or edge case

NEVER:

- Ask for email before results

- Require sensitive information

- Use confusing industry jargon

- Include trick questions



**4.3 Question Formulas**

**Formula 1: Behavioral**\
"When [situation], you typically..."

-   Answer A: [Behavior 1]

-   Answer B: [Behavior 2]

-   Answer C: [Behavior 3]

**Formula 2: Preference** "Which [thing] appeals most to you?"

-   Answer A: [Option with specific details]

-   Answer B: [Option with specific details]

-   Answer C: [Option with specific details]

**Formula 3: Priority** "What matters most when [context]?"

-   Answer A: [Value 1]

-   Answer B: [Value 2]

-   Answer C: [Value 3]

**Formula 4: Scenario** "You have [constraint]. What do you do?"

-   Answer A: [Approach 1]

-   Answer B: [Approach 2]

-   Answer C: [Approach 3]

**Formula 5: Identification** "Which statement best describes you?"

-   Answer A: [Identity 1]

-   Answer B: [Identity 2]

-   Answer C: [Identity 3]

**4.4 Answer Options**

Requirements per question:

- 3-4 answer choices (never 2, rarely 5+)

- Roughly equal answer length

- Mutually exclusive options

- No obvious "right" answer

- Specific, concrete language

- Positive framing

WRONG:

"I'm bad at marketing" (negative)

"I don't know" (throws off scoring)

"All of the above" (lazy design)

RIGHT:

"I prefer outsourcing to experts"

"I like to test and learn myself"

"I want a collaborative partnership"

// Pseudocode for result calculation

function calculateResult(answers) {

let scores = {

typeA: 0,

typeB: 0,

typeC: 0,

typeD: 0

};

// Weight questions differently

scores[answers[0]] += 10; // Q1: Standard weight

scores[answers[1]] += 15; // Q2: High weight (most important)

scores[answers[2]] += 15; // Q3: High weight

scores[answers[3]] += 10; // Q4-6: Standard weight

scores[answers[4]] += 10;

scores[answers[5]] += 10;

scores[answers[6]] += 5; // Q7-9: Low weight (preference)

scores[answers[7]] += 5;

scores[answers[8]] += 5;

scores[answers[9]] += 3; // Q10: Tiebreaker only

// Return highest scoring type

return Object.keys(scores).reduce((a, b) =>

scores[a] > scores[b] ? a : b

);

}

## ANTI-PATTERNS (Never Do These)

### Quiz Anti-Pattern 1: The Email Wall

\u274c **Wrong:** Requiring email BEFORE showing any questions

\u2705 **Right:** At minimum, show 2-3 questions before any gate

### Quiz Anti-Pattern 2: The Obvious Result

\u274c **Wrong:** Questions where the "correct" answer is obvious

\u2705 **Right:** Questions where all answers feel legitimate

### Quiz Anti-Pattern 3: The Generic Result

\u274c **Wrong:** Results that could apply to anyone

\u2705 **Right:** Results so specific the reader thinks "How did they know?"

### Quiz Anti-Pattern 4: The Hard Sell Result

\u274c **Wrong:** Result page that's just a sales page

\u2705 **Right:** Result page that provides value FIRST, then sells

### Quiz Anti-Pattern 5: The Shame Result

\u274c **Wrong:** Results that make people feel bad about their score

\u2705 **Right:** All results feel positive with opportunity framing

### Quiz Anti-Pattern 6: The Broken Progress Bar

\u274c **Wrong:** Progress bar that doesn't accurately reflect completion

\u2705 **Right:** Progress bar updates in real-time with each answer

### Quiz Anti-Pattern 7: The Desktop-Only Quiz

\u274c **Wrong:** Quiz that only works on desktop

\u2705 **Right:** Mobile-first design that works on any device
